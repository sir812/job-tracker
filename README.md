Job Tracker

A modern full-stack job application tracking platform built with React, TypeScript, Node.js, Postgre, Prisma, and Tailwind CSS.

This project was created to streamline the job and internship application process by providing a centralized platform for managing applications, tracking progress, organizing opportunities, and monitoring the hiring pipeline.

---

Project Overview

Job Tracker helps users keep track of job applications from submission to final outcome. Instead of managing applications through spreadsheets or notes, users can organize everything in a single dashboard.

The project follows a modern full-stack architecture with a React frontend, Node.js backend, PostgreSQL database, and Prisma ORM for database management.

This repository is currently under active development and serves as a portfolio project demonstrating full-stack engineering skills, database design, API development, authentication workflows, and modern frontend development practices.

Architecture

Frontend and backend are separated into independent services.

Client applications communicate with the backend through REST APIs.

The backend handles:

- Authentication
- Business logic
- Database interactions
- Request validation
- API responses

Prisma ORM acts as the bridge between the application and database.
